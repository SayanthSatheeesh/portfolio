# Sayanth Satheesh · Portfolio

> **Live Site:** [https://sayanthsatheeesh.github.io/portfolio/](https://sayanthsatheeesh.github.io/portfolio/)

A high-performance, single-page developer portfolio showcasing cloud infrastructure, full-stack projects, and cinematic UI/UX.

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, CSS3 (Modern Flex/Grid), ES6 JavaScript
- **Animations**: GSAP 3.12 (ScrollTrigger)
- **Smooth Scroll**: Lenis
- **Design Philosophy**: Glassmorphism, Cinematic Reveal, Minimalist Dark Mode

## ✨ Key Features
- **Deterministic UI**: Pixel-perfect responsive design from 375px to ultra-wide.
- **Immersive Micro-interactions**: Magnetic buttons, traveling nav dot, stats counters, and text scramble effects.
- **Accessibility & SEO**: Semantic HTML, ARIA labels, Open Graph meta tags, and `prefers-reduced-motion` compliance.
